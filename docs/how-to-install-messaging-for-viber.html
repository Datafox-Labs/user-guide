How to Install: Messaging for Viber

by DataFox Labs Team

Engage with your Viber™ users via Zendesk Suite (Sunshine Conversations). Unleash the full Zendesk power - with omnichannel routing, macros, triggers, ticket assignments and so much more!

Follow the steps below to proceed:

  1. Install Messaging for Viber App

  2. Launch the app in the Zendesk Navigation Bar

  3. Create a new conversations API and copy the App ID, Key ID and Secret key

  4. Return to the Messaging for Viber app and enter the conversation API details

  5. Proceed to create a Viber Bot Account and login to your Viber account

  6. Create a new Viber bot and copy the token ID (You may reuse a token ID of an existing Viber bot)

  7. Paste the Viber Public Account token in the Messaging for Viber App

  8. Click connect and the integration will be completed.
